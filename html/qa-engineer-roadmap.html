QA Roadmap UA 2026

Дорожная карта QA Engineer

Актуальный план обучения для старта в 2026.
Основано на требованиях Work.ua и DOU.

Что читать в 2026?

Классика: "Тестирование Дот Ком" (Роман Савин) — для легкого старта.
Серьезная база: "Тестирование ПО" (Святослав Куликов).
Совет: Учить термины сразу на английском. В украинских компаниях вся документация и Jira обычно на English.

Неделя 1: Фундамент (Теория)
Basis

То, что спросят на собеседовании в 100% случаев.

  • SDLC & STLC (Жизненные циклы)
  • Виды тестирования (Smoke, Regression, Sanity)
  • Пирамида тестирования (Unit -> Integration -> E2E)
  • Баг-репорт: Priority vs Severity (Критичность vs Важность)
Перейти к теории
Неделя 2: Документация & Jira
Practice

Учимся писать кейсы так, чтобы разработчик вас не проклинал.

  • Тест-кейсы и Чек-листы: отличия
  • Техники тест-дизайна: Граничные значения, Классы эквивалентности
  • Jira / Confluence (стандарт для Украины)
Изучить документацию
Неделя 3: Web Architecture & API
Technical

Самый важный технический блок.

  • Client-Server: Frontend vs Backend
  • HTTP методы (GET, POST, PUT, DELETE) и коды (200, 400, 500)
  • DevTools: Network (анализ запросов), Application (Cookies/Storage)
  • Postman: Коллекции и переменные
Погрузиться в API
Неделя 4: Базы данных (SQL)
Technical

Без SQL сейчас сложно пройти тех. собеседование.

  • Простые запросы: SELECT, WHERE, ORDER BY
  • Объединение таблиц: JOIN (Inner, Left, Right)
  • CRUD операции (Create, Read, Update, Delete)
Учить SQL
Неделя 5: AI, Git & CI/CD (Тренды 2026)
Must Have

То, что выделит тебя среди других джунов.

  • AI for QA: Генерация чек-листов через ChatGPT
  • Git: pull, push, merge, checkout
  • CI/CD Basics: Понимание пайплайнов GitLab/Jenkins
  • Английский: Уровень B1+ (Intermediate)
Современные инструменты
Неделя 6: Mobile Testing
Mobile

Основы тестирования приложений.

  • Android Studio & Xcode (Эмуляторы)
  • Снифферы: Charles Proxy / Fiddler (Перехват трафика)
  • Гайдлайны: Material Design vs Human Interface
Мобильное тестирование
